# Contribute

## Development

1. Fork the project and clone your fork
2. Install global dependencies

	``` bash
	npm install -g coffee-script
	```

3. Compile the project

	``` bash
	# Only once
	cake compile

	# On every change
	cake watch
	```

4. Run your tests

	``` bash
	cake test
	```

5. Prepare the project for publishing

	``` bash
	cake prepublish
	```

6. Publish the project

	``` bash
	cake publish
	```
